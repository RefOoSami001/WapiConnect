Check Sessions

GET /api/check-sessions

Get all WhatsApp sessions for the authenticated user.

Delete Session

DELETE /api/delete-session/{id}

Delete a WhatsApp session by ID.

Send Message

POST /api/send-message

Send a message to one or more WhatsApp numbers.

Get Contacts

POST /api/get-contacts

Get contacts from a WhatsApp session.

Get Group Members

POST /api/get-group-members

Get members of a specific WhatsApp group.

Create Auto-Reply Rule

POST /api/auto-reply-rules

Create a new auto-reply rule for a WhatsApp session.

Toggle Auto-Reply Rule

POST /api/auto-reply-rules/{id}/toggle

Toggle the active state of an auto-reply rule.

Delete Auto-Reply Rule

DELETE /api/auto-reply-rules/{id}

Delete an auto-reply rule.
